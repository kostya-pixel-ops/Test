Открой web/settings.py:

Шаг 3. Создание модели (таблицы)

Открой list_of_helps/models.py:

from django.db import models
 
class ListHelps(models.Model):
    name = models.CharField('Имя и фамилия героя', max_length=200)
    text = models.TextField('Информация о герое')
    foto = models.ImageField('Фото героя', upload_to='photo/%Y/%m/%d')
    data = models.DateField('Дата публикации', auto_now_add=True)
 
    def __str__(self):
        return self.name
 
    class Meta:
        verbose_name = 'Герой'
        verbose_name_plural = 'Список героев'

Шаг 6. Подключение шаблонов

В list_of_helps/views.py:

from django.shortcuts import render
from .models import ListHelps
 
def index(request):
    list = ListHelps.objects.all()
    return render(request, 'index.html', {'list': list})

Шаг 7. URLs

В web/urls.py:

from django.conf.urls.static import static

    path('admin/', admin.site.urls),
    path('', include('list_of_helps.urls')),
